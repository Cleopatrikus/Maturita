# Vytvorte program, ktorý vyskúša mladšieho súrodenca z malej násobilky. 
Program bude mať tieto vlastnosti:

- Program vygeneruje 10 príkladov na násobenie jednociferných čísel  a zapíše ich do textového súboru.
- Príklady nám postupne zadáva v textovom režime a prečíta našu odpoveď. Za správnu odpoveď získame 1 bod, ak odpovieme nesprávne 0 bodov.
- Po skončení riešenia program vypíše počet získaných bodov.
